# Proyecto: Cuentas Claras

App web para gestión de gastos personales/familiares.

## Stack
- React 19 + TypeScript + Vite
- Framer Motion (animaciones)
- Lucide React (iconos)
- jsPDF + html2canvas (exportar PDF)
- @google/genai (IA de Google)

## Comandos
- `npm run dev` — servidor de desarrollo
- `npm run build` — compilar producción
- `npm run preview` — previsualizar build

## Estructura
- `src/components/` — componentes React
- `src/hooks/` — hooks personalizados
- `src/context/` — contextos React
- `src/utils/` — utilidades

## Reglas del proyecto
- Usar TypeScript estricto, no `any`
- Componentes funcionales con hooks, no clases
- Estilos con Tailwind CSS (clases inline)
- No agregar dependencias sin preguntar primero
- Responder errores de TypeScript antes de ejecutar
